YAT library

YAT is a general purpose C++ library.
Its purpose is to provide the developer with useful, easy to use and powerful C++ tools for the common (i.e. classical) developement tasks.
The topics currently addressed by YAT library are:

  • Generic containers
  • Bit streams
  • File access
  • Memory management
  • TCP and UDP socket client
  • Plugins
  • System utilities
  • Threading
  • Time utilities
  • General Utilities
Remarks
Yat4Tango is a specialization - and also an extension - of the YAT library for the TANGO devices developers. For instance, the yat4Tango::DeviceTask is better adapted to the context of TANGO Device than the yat::Task.
